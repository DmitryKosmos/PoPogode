Приложение используется для вывода информации о текущей погоде и рекомендаций одежды.

оно пока состоит из 4-х файлов: clothe.py, weather.py, messages.py и main.py

clothe.py - библиотека с типами одежды
weather.py - библиотека отвечающая за получение данных о погоде
messages.py - библиотека с сообщениями, которые будут выводиться в зависимости от погодных условий
main.py - главный файл, в ктором всё собрано вместе

В дальнейшем будет реалзована функция добавления своей одежды с фото и графический интерфейс
